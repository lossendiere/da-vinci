Introduction

Le logiciel de montage Da Vinci Revolve est un logiciel gratuit utilisé par des monteurs professionnels pour la télévision et le montage de films. Si vous êtes débutants, il est possible d'utiliser seulement les fonctions basiques qui sont suffisantes pour les petits projets.

Donner un nom au projet

Le nom que vous avez choisi apparaît à deux endroits en haut à gauche de la fenêtre et plus bas à droite

Création du projet

Quand le logiciel est démarré il suffit de cliquer sur "New Project" ( 1 )
2 ) Si vous avez beaucoup de projets, il est possible de créer des dossiers ( New Folder ) pour les ranger dedans.
3 ) Il est possible d'afficher les projets sous forme de vignettes ou listes
4 ) De faire une recherche pour trouver un projet en tapant son nom
5 ) Pas la peine de cliquer ici, cela permet de créer des bases de données et c'est utile seulement pour les projets professionnels.

Les étapes d'un projet

Pour un projet basique vous devez suivre 3 étapes :
1 - Sélectionnez les médias ( les rush )
2 - Faire le montage ( placer les rush sur la ligne de temps )
6 - Adapter la vidéo à l'appareil ou le service internet sur lequel sera diffusé la vidéo ( Cinéma, DVD, Youtube ... )
Si le cahier des charges vous impose des critères de niveau professionnel
3 - vous pouvez créer des effets spéciaux avec le module fusion
4 - Vous pouvez changer les couleurs ( Colorimétrie )
5 - vous pouvez créer des montage et mixages audio professionnel avec le module Fairlight

Intéret de la colorimétrie

La colorimétrie permet d'avoir par exemple une couleur du film qui ne dépende pas de la météo le jour du tournage.
Des plans successifs d'une même scène se déroulant dans le film le même jour peuvent être tournés à des jours différents, donc avec une météo changeante.

La colorimétrie peut être aussi un choix artistique.
Exemples : Le fabuleux destin d’Amélie Poulain, un long dimanche de fiançailles, au revoir la haut.

Import des Rush

  • Import classique
  • Import avancé avec triage des rush

Cette méthode est la méthode utilisée par les logiciels de montage basiques, l'ensemble du rush est importé dans la partie montage. Vous pouvez utiliser cette méthode mais seulement pour les projets simples avec des rushs courts et pas beaucoup de rushs.
Imaginez vous avec des rush très long, vous allez avoir une timeline très longue pour une seule petite partie interessante et vous allez perdre beaucoup de temps.

Pour cela il faut cliquer sur :
1 )  Fichier
2 ) Puis Importer un fichier
3 ) Enfin importer des médias ( vous pouvez importer plusieurs médias en même temps en sélectionnant plusieurs fichiers )

Il faut au préalable indiquez des points de montage pour vos espaces de stockages.
C'est a dire des emplacements ( des dossiers ) de disques dur, cartes SD, stockages réseau ... ou sont stockés vos ruhs.
Ceci va permettre par la suite de trier rapidement vos rush et prendre seulement la partie de vos rush qui vous interesse.

Cela se passe dans les préférence "Système" de Da Vinci Resolve.
Si vous cochez "Visualiser automatiquement les espaces de stockage locaux et en réseau", tous les disques dur, clef USB, cartes SD, lecteurs réseaux vont apparaitres automatiquement dans vos espaces de stockage, ce qui peut faire une très longue liste. Il est préférable selon moi de faire apparaitre seulement les emplacements ou vous savez qu'il y a des vidéos.

Attention !! vérifiez bien en bas que vous êtes dans la partie média si vous ne voyez pas vos disques dur dans la partie "espaces de stockage" c'est que vous n'êtes dans la partie montage au lieu de la partie média par exemple.

Les vidéos ( 2 )  des espaces de stockage ( 1 ) apparaissent sous forme de vignettes, en approchant la souris des vignettes, une prévisualisation apparait en zone (3) ainsi qu'une prévisualisation sonore en zone ( 5 )
Le but est de faire glisser les rush ou extraits de rush de la zone ( 2 ) à la zone ( 5 ).
Les vidéos tombent dans ce qu'on appelle un chutier appellé aussi en Anglais ( Bin ).
Un chutier est une réserve de rush ayant une cohérence, un peu comme on fait tomber des vis de même type dans différents tiroirs. Le chutier par défaut est le chutier "Master"

Ranger ses rush dans des chutiers

Un simple clic droit dans la zone Master à gauche permet de créer de nouveaux chutiers.
Les chutiers à droite apparaissent comme des dossiers dans lesquels on peut faire glisser des vidéos.
Pour supprimer des rush et des chutiers, il faut les sélectionner et appuyer sur la touche suppression du clavier.
